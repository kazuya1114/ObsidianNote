回帰モデルは、給料といった値を予測する際に使われる。  
例えば、独立変数が時間だとすると、回帰モデルを作成することによって将来のある値を予測することになる。  
回帰の手法は、線形回帰からサポートベクトル回帰、ランダムフォレストまで多くの種類がある。  

# 単回帰分析

![単回帰分析](../pic/2.png)
4年目の年収を予測するには、判明しているデータ（今回だったら5つの点）に一番近い直線（=回帰直線）を引き、X軸・Y軸の接点の値が予測値となる。  
`y = b0 + b1X`
b0：y軸との切片の値  
b1：就業年数が1年増えたら年収がどれだけ増えるか（係数）
データ：年収  
アルゴリズム：直線をどのように引くか

## 回帰直線を求めるアルゴリズム
### 最小二乗法

![最小二乗法](../pic/3.png) 
式：`i∑​(yi​−y^​i​)2`  
∑(yi​−(axi​+b))2（=誤差）を最小にする **a（傾き）と b（切片）を数学的に直接求める方法**。

### 勾配降下法

∑(yi​−(axi​+b))2（=誤差）を最小にする **a と b を求める方法**だが、最小二乗法とは異なり、初期値からスタートし傾きを見ながらパラメータを少しずつ更新して最適な a（傾き）とb（切片）に近づけるという **最適化アルゴリズム**。

## 単回帰分析の実装

```python
# 単回帰分析
import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

# 分析前の前処理
dataset = pd.read_csv('data/Salary_Data.csv')
X = dataset.iloc[:, :-1].values  # 独立変数（YearsExperience列）
Y = dataset.iloc[:, -1].values   # 従属変数（Salary列)
X_train, X_test, Y_train, Y_test = train_test_split(X, Y, test_size=1/3, random_state=0) # データ分割

# 単回帰分析の実行（データセットを使った学習）
regressor = LinearRegression()
regressor.fit(X_train, Y_train) # 学習（回帰直線の決定（＝切片と傾きの決定））

# テストデータを使った予測
y_pred = regressor.predict(X_test) # 予測値の算出

# 結果の可視化
plt.scatter(X_train, Y_train, color='red') # 学習用データの散布図を描画
plt.plot(X_train, regressor.predict(X_train), color='blue') # 回帰直線を散布図に描画
plt.xlabel('Years of Experience')
plt.ylabel('Salary')
plt.show() # 学習用データのプロット表示

# テスト用データの可視化
plt.scatter(X_test, Y_test, color='red') # テスト用データの散布図を描画
plt.plot(X_train, regressor.predict(X_train), color='blue') # 回帰直線を散布図に描画
plt.xlabel('Years of Experience')
plt.ylabel('Salary')
plt.show() # テスト用データのプロット表示

# 切片（b0）の取得
intercept = regressor.intercept_ # 切片（b0）の取得

# 傾き（b1）の取得
slope = regressor.coef_ # 傾き（b1）の取得(配列で返される)
slope_value = regressor.coef_[0] # 傾き（b1）の取得(値として取得)

# 就業年数が12年の場合の給与を予測
years = 12
predicted_salary = regressor.predict(np.array([[years]])) # 就業年数が12年の場合の給与を予測
```

# 重回帰分析

単回帰分析と比べ次元（独立変数）が増えている
y = b0 + b1X1 + b2X2 + b3X3  + ... biXi

## 線形回帰の前提

1. 線形性
2. 内生性がないこと
3. 正規性と同分散性
4. 自己相関がないこと
5. 多重共線性がないこと

## P値

統計学で使用される指標（ある事象が起こる確率）。

## 独立変数を抽出する理由

1. モデルをシンプルにするため
2. 定性的な評価を行えるようにするため

## 変数の選択方法①（All-in）

すべてを独立変数にする

【使う場面】
- 事前情報がない場合（どれを独立変数として扱えばモデルがシンプルとなるかわからない）
- All-inで分析しなければいけない場合（すべて使用することを指定された場合など）

## 変数の選択方法②（Backward Elimination（変数減少法））

その独立変数がyの値を予測するうえで重要ではない独立変数を探し、取り除く。
全ての独立変数に対して有意水準よりもp値が小さくなったらモデル完成（重要な独立変数のみが残った状態）

step1：モデルをそのままにするための閾値として有意水準を決める
step2：全ての独立変数を使ってモデルを作成する
step3：p値が最も高い独立変数を見つけ、そのp値が有意水準よりも大きければstep4へ、小さければモデルは完成
step4：その独立変数を取り除く
step5：取り除いたうえで改めてもモデルを作成する（step3へ）

## 変数の選択方法⓷（Forward Selection（変数増加法））

step1：モデルにその変数を入れるための閾値としての有意水準を決める
step2：全ての独立変数それぞれにおいて最もp値が低い変数を求める（対象項目が100個あるならモデルを100個作成して比較）
step3：曽於変数を追加した上で、改めてそれぞれの独立変数においてモデルを作成する
step4：その中で、p値が最も低い独立変数を見つけ、その値が有意水準よりも小さければモデルにその変数を入れ、それがなければモデルを完成させる

## 変数の選択方法④（Bidirectional Elimination（変数増減法））

step1：モデルにその変数を入れる・取り除くための閾値としての有意水準を決める
step2：Forward Selectionを行う
step3：Backward Eliminationを行う
step4：取り除く、もしくは追加する変数がなくなった時点でモデルを完成させる

## 変数の選択方法⑤（All Possible Models）

演算の負荷が高い

step1：モデルの当てはまりの良さを評価するための基準を選択する（赤池情報量基準など）
step2：全ての取りえる変数において回帰モデルを作成する
step3：最もスコアが高いモデルを採用する

## 重回帰分析の実装

```python
# 重回帰分析
import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import OneHotEncoder

# 実装方法は単回帰分析と同様。ただし、独立変数が複数あるため、データセットの前処理でカテゴリ変数のエンコーディングを行う必要がある。

# 分析前の前処理
dataset = pd.read_csv('data/50_Startups.csv')
X = dataset.iloc[:, :-1].values  # 独立変数（YearsExperience列）
Y = dataset.iloc[:, -1].values   # 従属変数（Salary列)

# カテゴリ変数のエンコーディング
ct = ColumnTransformer(transformers=[('encoder', OneHotEncoder(), [3])], remainder='passthrough') # state列をワンホットエンコーディング
X = np.array(ct.fit_transform(X)) # fitとtransformを同時に実行（Xが指定されたため、Country～Salary列）。np.arraayでndarray型に変換（基本的には不要。おまじない）

# 訓練用データとテスト用データセットへの分割
# 　test_size=0.2：データ全体の20%をテスト用データに使用（残りの80%を訓練用データに使用）
# 　random_state=0：乱数シードを0に設定（毎回同じ分割結果になるようにするための設定）
X_train, X_test, Y_train, Y_test = train_test_split(X, Y, test_size=0.2, random_state=0)

# モデルの学習
regressor = LinearRegression()
regressor.fit(X_train, Y_train) # 学習（回帰直線の決定（＝切片と傾きの決定））

y_pred = regressor.predict(X_test) # 予測値の算出
np.set_printoptions(precision=2) # 小数点以下2桁まで表示
np.concatenate((y_pred.reshape(len(y_pred), 1), Y_test.reshape(len(Y_test), 1)), 1) # 予測値と実際の値を並べて表示
```

# 多項式回帰

単回帰：y = b0 + b1X1
多項式回帰：y = b0 + b1X1 + b2X2^2

![多項式回帰](../pic/4.png)

## 多項式回帰の実装

```python
# 多項式回帰分析
import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import OneHotEncoder
from sklearn.preprocessing import PolynomialFeatures

# 分析前の前処理
dataset = pd.read_csv('data/Position_Salaries.csv')
X = dataset.iloc[:, 1:-1].values  # 独立変数（YearsExperience列）
Y = dataset.iloc[:, -1].values   # 従属変数（Salary列)

# モデルの学習（単回帰）
lin_reg = LinearRegression()
lin_reg.fit(X, Y)

# モデルの学習（多項式回帰）
poly_reg = PolynomialFeatures(degree=2) # degree=2：2次の多項式回帰を指定
X_poly = poly_reg.fit_transform(X) # Xを多項式特徴量に変換
lin_reg_2 = LinearRegression() # 多項式回帰のモデルを作成
lin_reg_2.fit(X_poly, Y) # 学習（回帰曲線の決定（＝切片と傾きの決定））

# 結果の可視化
plt.scatter(X, Y, color='red') # 学習用データの散布図を描画
plt.plot(X, lin_reg.predict(X), color='blue') # 単回帰の回帰直線を散布図に描画
X_grid = np.arange(min(X), max(X), 0.1) # Xの最小値から最大値まで0.1刻みの数値を生成
X_grid = X_grid.reshape(len(X_grid), 1) # X_gridを列ベクトルに変換
plt.scatter(X_grid, lin_reg_2.predict(poly_reg.fit_transform(X_grid)), color='green') # 多項式回帰の回帰曲線を散布図に描画
plt.title('Truth or Bluff (Polynomial Regression)')
plt.xlabel('Position level')
plt.ylabel('Salary')
plt.show() # 学習用データのプロット表示

# 就業年数が6.5年の場合の給与を予測
years = 6.5
predicted_salary_lin = lin_reg.predict(np.array([[years]])) # 単回帰で就業年数が6.5年の場合の給与を予測
predicted_salary_poly = lin_reg_2.predict(poly_reg.fit_transform(np.array([[years]]))) # 多項式回帰で就業年数が6.5年の場合の給与を予測
```


# サポートベクトル回帰（SVR）

![サポートベクトル回帰](../pic/5.png)

## サポートベクトル回帰の実装

```python
# サポートベクトル回帰分析（SVR）
import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import OneHotEncoder
from sklearn.preprocessing import PolynomialFeatures
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVR

# 分析前の前処理
dataset = pd.read_csv('data/Position_Salaries.csv')
X = dataset.iloc[:, 1:-1].values  # 独立変数（YearsExperience列）
Y = dataset.iloc[:, -1].values   # 従属変数（Salary列)
Y = Y.reshape(len(Y), 1) # yを列ベクトルに変換

# フィーチャースケーリング（特徴量の標準化）
sc = StandardScaler()
X_train = sc.fit_transform(X) # Xの標準化
Y_train = sc.fit_transform(Y) # Yの標準化

# モデルの学習
regressor = SVR(kernel = 'rbf') # カーネル関数にRBF（Radial Basis Function）を指定
regressor.fit(X_train, Y_train.ravel()) # 学習（回帰曲線の決定（＝切片と傾きの決定））

# 結果の予測
year = 6.5
sc.inverse_transform(regressor.predict(sc.transform(np.array([[year]])))) # 就業年数が6.5年の場合の給与を予測（予測値は標準化されているため、逆変換して元のスケールに戻す）

# 結果の可視化
X_grid = np.arange(min(sc.inverse_transform(X_train)), max(sc.inverse_transform(X_train)), 0.1) # Xの最小値から最大値まで0.1刻みの数値を生成
X_grid = X_grid.reshape(len(X_grid), 1) # X_gridを列ベクトルに変換
plt.scatter(sc.inverse_transform(X_train), sc.inverse_transform(Y_train), color='red') # 学習用データの散布図を描画
plt.plot(sc.inverse_transform(X_grid), sc.inverse_transform(regressor.predict(X_grid)), color='blue') # 回帰曲線を散布図に描画
plt.title('Truth or Bluff (SVR)')
plt.xlabel('Position level')
plt.ylabel('Salary')
plt.show() # 学習用データのプロット表示
```
