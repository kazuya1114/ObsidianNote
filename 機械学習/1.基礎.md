# 機械学習、ディープラーニング、AI（人工知能）の違い

### 人工知能

```
コンピュータが人間のような知能を模した形で、意思決定、テキスト処理、画像認識などができるようにすることを言う。
また、人工知能は広範な分野を示した言葉であり、その中に機械学習やロボティクスといった内容も含まれている。
```

### 機械学習

```
機械学習は人工知能に含まれる考え方であり、試行を繰り返すことで機会があるタスクを改善できるようにする仕組み、ということができる。
重要なポイントとして、あらゆる機械学習の手法というのは人工知能に属するということができるが、人工知能の手法の全てが機械学習に属するとは言えない。
なぜなら、人工知能の一部というのはあるルールに基づいて学習を行うという機械学習の考え方を逸脱している部分があるため。
```

### ディープラーニング

```
ディープラーニングは、画像やテキストといった非常に大きいデータを使ってニューラルネットワークのモデルを主に活用しているという意味において、機械学習の中の特別な分野と言うことができる。

ニューラルネットワークは人間の脳に着想を得たモデルである。人間の脳は数十億のニューロンがあり、そのニューロンはそれぞれが関係し合いながら電気信号などを送ることによってお互いに連携しあい、見たり聞いたりすることを可能としている。

そして、ニューラルネットワークは人工的なニューロンを繋げていき、層を作って演算を行うことで機能する。

ディープラーニングと機械学習の違いは、以下の例でより具体的に表現することができる。

-機械学習　１学習をさせるためのモデルを選択、２人の手によって特徴を抽出

-ディープラーニング　１ネットワークの型を決める、２演算が行われた結果、自動的に特徴が抽出される
```

# 機械学習の実生活への応用例

1. FaceBook（畳み込みニュートラルネットワーク、トンプソンサンプリング）
2. ゲーム（Random Forest）
3. VR
4. iPhoneの音声認識
5. ロボット（強化学習）
6. Amazon、Netflix、audible（商品購入の傾向）
7. 悪い状態、いい状態を学習させてAIに判断させる
8. 天気予報（過去の傾向から分析の精度を高めるj）
