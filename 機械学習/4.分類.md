
# ロジスティック回帰

******ある事象が起こる確率を予測するための機械学習手法**で、主に**分類問題**に使う。

名前に「回帰」とあるが、目的は

> **データをクラス（例：0か1）に分類すること**

![ロジスティック回帰](../pic/11.png)
![ロジスティック回帰2](../pic/12.png)

## 計算式

$$y = B₀ × B₁^{x₁} × B₂^{x₂}$$

$$ý = \frac{1}{1 + e^{-z}}$$$$z = b₀ + b₁x₁ + b₂x₂$$

## 分類分け例

b₀ + b₁x₁ + b₂x₂ > 0：生活習慣病
b₀ + b₁x₁ + b₂x₂ <= 0：健康

## フィーチャースケーリング（特徴量の標準化）実行の判断基準

最も簡単な判断基準は

> **距離・勾配を使うアルゴリズム → スケーリング必要**
> **木ベースアルゴリズム → 不要**

| アルゴリズム         | フィーチャースケーリングの要否 | 理由     |
| -------------- | --------------- | ------ |
| ロジスティック回帰      | 要               | 勾配降下法  |
| SVM / SVR      | 要               | 距離計算   |
| KNN            | 要               | 距離計算   |
| ニューラルネット       | 要               | 勾配最適化  |
| PCA            | 要               | 分散ベース  |
| 線形回帰（L1/L2正則化） | 要               | 係数の影響  |
| 決定木            | 否               | 条件分岐のみ |
| 回帰木            | 否               | 同上     |
| ランダムフォレスト      | 否               | 木ベース   |
| XGBoost        | 否               | 木ベース   |
| LightGBM       | 否               | 木ベース   |
